# 🏪 CasaVidal - Frontend

Interfaz web moderna para el sistema CRM de gestión de ferretería.

## 🚀 Tecnologías

- **React 18** - Biblioteca UI
- **TypeScript** - Tipado estático
- **Vite** - Build tool ultrarrápido
- **TailwindCSS** - Estilos utilitarios
- **Zustand** - Gestión de estado
- **Axios** - Cliente HTTP
- **React Hook Form** + **Zod** - Formularios y validaciones
- **Lucide React** - Iconos

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/tavoidkk/casavidal-frontend.git
cd casavidal-frontend

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env
# Edita VITE_API_URL con la URL de tu backend

# Iniciar servidor de desarrollo
npm run dev
